Box Model Comparison

All three boxes below have width: 250px, padding: 20px, and a 4px border — but render at different total widths.

box-sizing: border-box (default in this demo's * rule) — total rendered width stays 250px.
box-sizing: content-box — padding & border are ADDED to 250px, so this box is actually 250+40+8 = 298px wide.
Same as the first box, showing margin creates the gap you see between these boxes.
